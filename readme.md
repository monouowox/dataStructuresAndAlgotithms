## 数据结构与算法
* `B` 数据结构：数据在存储空间中的组织形式
* `B` 算法：解决问题的方式
常见的数据结构
 * `A` 数组--存储空间中连续的一段空间，扩容换一个更大的空间。增删麻烦（空间内容需要位移），查找修改已存在的内容方便（（下标）指针直接找到）。
 * `A` 队列--两端分前后，只许后端添加元素，前端删除元素，先进先出。
    --普通队列
    --优先级队列：比起普通队列就是可以插队（根据优先级）。例如：头等舱优先登记、军人优先、急诊。
        就是数据有优先级
 * `A` 栈----只许栈顶增加和删除元素。增删对应了进栈和出栈，特点先进后出。函数调用就会使用。
 * `A` 堆
 * `A` 树
 * `A` 图
 * `A` 链表--空间存了值和另一个空间的指针不使用一段空间，增删容易，查找不便（线性查找）。
 * `A` 散列表
 * `A` 集合----数据无序（无法通过下标值获取元素），数据不重复
 * `A` 字典----一一对应的关系，根据key查找到值，key不可以重复
 * `A` 哈希表----基于数组，优化数组不足，查找效率比树更快，但数据无序。
